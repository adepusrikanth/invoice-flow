Accessibility Rules

•  Never use Primary (#6367FF) as text on colored backgrounds — only on white or Neutral 50

•  Neutral 400 (#9494AB) is only WCAG AA for large text (18px+). Use Neutral 500 for small text

•  Always provide focus rings (3px Primary at 20% opacity) for keyboard navigation

•  Never rely on color alone to convey meaning — always pair with icons and/or text labels

•  Test all color combinations with a color blindness simulator (Deuteranopia, Protanopia, Tritanopia)

•  Dark mode: invert Neutral scale. Primary stays the same (sufficient contrast on dark backgrounds)
